HTTP
-Es el protocolo de comunicación que permite la transferencia de información en la World Wide Web. Define cómo se transmiten y se formatean los datos entre un cliente (como un navegador web) y un servidor (donde se alojan los sitios web y sus recursos).
HTML
-Es el lenguaje estándar utilizado para crear páginas web. Se utiliza para estructurar y presentar contenido en la web mediante etiquetas que definen elementos como encabezados, párrafos, enlaces, imágenes, entre otros.
URL
-Es la dirección de un recurso en internet. Define la ruta para acceder a recursos como páginas web, imágenes, archivos, etc. Una URL típica tiene varios componentes, como el protocolo (http:// o https://), el nombre de dominio (www.ejemplo.com), la ruta y el nombre del recurso.
TCP/IP
-Es el conjunto de protocolos de comunicación que permite la conexión de dispositivos en redes de internet. TCP (Protocolo de Control de Transmisión) se encarga de dividir los datos en paquetes para su envío y garantizar que lleguen correctamente, mientras que IP (Protocolo de Internet) se encarga de la dirección y el enrutamiento de estos paquetes a través de la red.
Internet
-Es una red global de redes interconectadas que utilizan el protocolo TCP/IP para permitir la comunicación y el intercambio de información entre dispositivos en todo el mundo. Internet permite el acceso a una amplia gama de recursos y servicios, como páginas web, correos electrónicos, transferencia de archivos, entre otros.
W3C
-Es una organización internacional que desarrolla estándares y pautas para la web con el objetivo de garantizar su crecimiento a largo plazo. El W3C establece recomendaciones para tecnologías web como HTML, CSS, entre otros, con el fin de asegurar la interoperabilidad y la accesibilidad en la web.
Mozilla Foundation
-Es una organización sin fines de lucro que impulsa el desarrollo de tecnologías abiertas y estándares web. Es conocida por ser la creadora del navegador web Firefox y ha contribuido significativamente al desarrollo de estándares abiertos, herramientas y recursos para la web.
